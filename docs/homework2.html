Homework: Dataset and Distribution Analysis

Using Microsoft Access and SQL queries, exploring univariate and bivariate distributions.

1. Concept of Dataset and Distribution

A dataset is a collection of structured data, often organized in tables with rows (records) and columns (variables). It captures observations such as student subjects, study hours, and grades in this homework.

A distribution describes how values of a variable are spread. Univariate distribution examines one variable at a time (e.g., grades frequency). Bivariate distribution explores the relationship between two variables (e.g., study hours vs. grades).

3. Univariate Distributions

Counts and percentages for one variable at a time using SQL queries in Access.

a. Subject Distribution

SELECT subject, COUNT(*) AS frequency,
  ROUND((COUNT(*) * 100.0) / (SELECT COUNT(*) FROM HMWK2), 2) AS percentage
FROM HMWK2
GROUP BY Subject
ORDER BY percentage DESC;

4. Bivariate Distribution

Binned relationship of study_hours and grade showing counts of students in hour and grade ranges using Access SQL:

3. Bivariate Distribution Query (Access SQL)

SELECT
  SWITCH(
    study_hours BETWEEN 0 AND 2, '0-2',
    study_hours BETWEEN 3 AND 5, '3-5',
    study_hours BETWEEN 6 AND 8, '6-8',
    TRUE, 'Other'
  ) AS hours_bin,
  SUM(IIF(grade BETWEEN 5 AND 6, 1, 0)) AS [5-6],
  SUM(IIF(grade BETWEEN 7 AND 8, 1, 0)) AS [7-8],
  SUM(IIF(grade BETWEEN 9 AND 10, 1, 0)) AS [9-10],
  COUNT(*) AS total
FROM HMWK2
GROUP BY
  SWITCH(
    study_hours BETWEEN 0 AND 2, '0-2',
    study_hours BETWEEN 3 AND 5, '3-5',
    study_hours BETWEEN 6 AND 8, '6-8',
    TRUE, 'Other'
  )
ORDER BY
  SWITCH(
    study_hours BETWEEN 0 AND 2, '0-2',
    study_hours BETWEEN 3 AND 5, '3-5',
    study_hours BETWEEN 6 AND 8, '6-8',
    TRUE, 'Other'
  );

This query groups students by study hour bins and counts the number with grades in each range. Each row shows one hours bin, each column a grade bin.
